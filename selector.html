declare

expression:= selector: value

selector: year | month | date | day | not | yesterday | range

value: Number | String | Range | Others

AndExpression:= expression ((&& | and)? expression)?

OrExpression:= AndExpression ((|| | or) AndExpression)?

year selector

The value of this selector is Number or leap-year. If the value is Number, this selector matches the year the value implies. If the value is leap-year, this selector matches leap year.

Example

year: 2000
year: leap-year

month selector

The value of this selector is Number expressing the month that you want to match. The value cannot be less than 1 or greater than 12. Error will occur.

Example

month: 1

date selector

The value of this selector is Number, vernal-equinox-day, autumnal-equinox-day or full-moon-night. When the value is Number, that cannot be less than 1 or greater than 31.

Example

date: 31
date: vernal-equinox-day

day selector

The value of this selector is sunday, monday, tuesday or String of other days. You can use abbreviations such as sun, mon or tue. You can use ordinal number. Putting ordinal number such as 1st and one space or one hyphen.

Example

day: sunday
day: 1st-wed
day: 2nd friday

not selector

The value of this selector is expression. The value can accepts any selector even if the value is not selector expression.

Example

not: month: 1

range selector

The format of the value of this selector is year/month/date...year/month/date. mnonth accepts only 1 through 12. date accepts only 1 through 31. the splitter "..." can be "..". You can omit start date or end date.

Example

range: 1901/1/1...2000/12/31 range: ...2000/12/31

Expression Example

date: 13 day: 3rd-wednesday
date: 13
day: wednesday
month: 5 date: 5
year: leap-year month: 2 date: 29
year: leap-year month: 5
